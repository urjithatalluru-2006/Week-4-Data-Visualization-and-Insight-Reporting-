# Week 4: Data Visualization and Insight Reporting

## Objective
Analyze automotive data using visualizations and derive meaningful business insights.

## Dataset
Automobile Price Dataset containing 211 records and 25 features.

## Visualizations Created
1. Average Price by Car Body Type
2. Horsepower vs Price Scatter Plot
3. Correlation Heatmap
4. Fuel Economy vs Price Analysis

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- VS Code

## Key Insights
- Higher horsepower vehicles tend to have higher prices.
- Engine-related features strongly influence price.
- Body style impacts average vehicle pricing.
- Fuel efficiency helps identify different market segments.
